## Current mathematical focus

The active line of research is this experiment family:

Given a rational transform, especially one involving:
- multiply by a rational number
- divide by a rational number
- take floor / integer truncation

study the local jump behavior at prime indices.

The first active example is built around the reciprocal pair:
- 1.6 = 8/5
- 0.625 = 5/8

Key identity:
- divide by 0.625 = multiply by 8/5 = multiply by 1.6

## Core discovery from the first experiment

The user’s original hand experiment was:

- take integers from 1 upward
- divide by 0.625
- examine the integer-style output
- isolate prime inputs
- look at the local step pattern before and after each prime

What this really is mathematically:
- bc was truncating / behaving in an integer-style way in the original hand workflow
- the experiment is captured by:
  - f(n) = floor( n / 0.625 )
  - equivalently f(n) = floor( 8n / 5 )

For each prime p, the local pair recorded is:
- jump_in = f(p) - f(p - 1)
- jump_out = f(p + 1) - f(p)

So a prime gets tagged by a pair such as:
- 2 1
- 1 2
- 2 2

This is now implemented in Rust and verified.

## Deep interpretation of what the current result likely means

The current floor-jump pattern does appear to be real and reproducible.

However, the strongest current interpretation is:

### The visible pair structure is primarily generated by:
- rational multiplication
- floor/truncation
- denominator periodicity
- modular residue classes

and only secondarily by:
- where primes happen to fall inside those periodic cycles

For `floor(8n/5)`, the jump pattern is periodic with denominator 5 influence.
For `floor(5n/8)`, the jump pattern is periodic with denominator 8 influence.

This means the present work is not yet evidence of a new deep prime law.
It is better understood as:

> a study of how primes sample deterministic floor-sequence backgrounds

That is still worthwhile, because:
- it teaches exact arithmetic discipline
- it avoids false hype
- it may reveal prime-sensitive deviations once the background is properly subtracted

The big long-term research question is:
- after removing the transform-generated periodic structure, is there any residual prime-specific signal that remains interesting?

That is the serious path forward.

## Current honest position regarding the Riemann Hypothesis

Important carry-over doctrine:

The current experiments are **not** a proof path to RH in any direct sense.

Why:
- RH is a global analytic statement about zeta zeros
- finite computational patterns in floor sequences do not by themselves imply RH

However, these experiments could still become useful if they:
- reveal robust prime-sensitive invariants
- isolate deviations from expected modular background
- lead to a clean conjecture that can be phrased mathematically
- connect later to prime counting error terms, Chebyshev functions, Möbius behavior, or other RH-adjacent quantities

So the correct posture is:

- current work is exploratory and structural
- current work is likely modular/floor arithmetic first
- future work may become RH-adjacent only if the signal survives baseline subtraction and large-scale testing

This distinction must remain explicit in future sessions.
